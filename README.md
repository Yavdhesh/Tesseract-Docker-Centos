# Tesseract Docker Centos

Namastey, It is ready to go and use docker image for centos. 

### Author: Yavdhesh Sanchihar


## Build


```bash
docker build -t tesseract-centos .
```

## Usage

```python
docker container run -d -p 5000:5000 tesseract-centos 
```
## Run locally
```python
http://localhost:5000/
```
